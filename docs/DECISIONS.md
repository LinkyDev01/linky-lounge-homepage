## 상시 원칙 (만료 없음)

- **CTA 하단 고정 방식은 절대 변경 금지** — 문구·내용만 조정 가능
- **자잘한 변경은 재확인 없이 즉시 배포 허용** ("자잘한 거니까 바로 배포해") — 단 '자잘한'의 경계와 예외(모호 해석 구현 등)는 `lazyday-deploy` 스킬 §0 기준을 따름
- **지시의 적용 범위는 직전 논의 대상에 한정** — "그 외는 건들지 마"
- **가격 등 참가 상세는 인터뷰 페이지에서만** (랜딩·신청 폼 노출 금지)
- **섹션 간 여백 금지** (이어 붙임) + 섹션 내부 콘텐츠 아래 하단 패딩 확보
- **브랜드 카피는 운영자 원고 그대로** — AI 임의 창작 금지
- **디자인 개편 지시에는 단일 개선안 금지** — 레퍼런스 기반 복수 시안 쇼케이스로 응답 (선례: preview/faq-designs). 제안 단계도 '진행중'으로 즉시 기록
